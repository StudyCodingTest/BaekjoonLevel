# 이항 계수 1

Step07 [이항 계수 1](https://www.acmicpc.net/problem/11050)

## 문제

자연수 N과 정수 K가 주어졌을 때 이항 계수 (NK)\(\binom{N}{K}\)를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 N\(N\)과 K\(K\)가 주어진다. (1 ≤ N\(N\) ≤ 10, 0 ≤ K\(K\) ≤ N\(N\))

## 출력

 (NK)\(\binom{N}{K}\)를 출력한다.

## 예제 입력 1 

```
5 2
```

## 예제 출력 1 

```
10
```
